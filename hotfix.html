Project Data
Projects are used to organize a series of data points (Units) under a single umbrella. For instance, each project could represent a single set of experiments or a set of related experiments.
Units
Units are single data points, e.g., single files, that are used in further analysis. Units may be primary, typically refering to user-uploaded initial data, or secondary, usually referring to data generated from an anaylsis and linked to a primary Unit.
